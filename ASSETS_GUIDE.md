# Assets Guide

For å fullføre appen, må du legge til følgende bilder i `assets/`-mappen:

## Påkrevde bilder

### 1. icon.png
- **Størrelse**: 1024x1024 piksler
- **Format**: PNG
- **Bakgrunn**: Transparent eller solid farge
- **Beskrivelse**: App-ikonet som vises på hjemmeskjermen

### 2. splash.png
- **Størrelse**: 1242x2436 piksler (eller større)
- **Format**: PNG
- **Bakgrunn**: #2E7D32 (organisasjonens grønne farge)
- **Beskrivelse**: Splash-skjermen som vises ved oppstart

### 3. adaptive-icon.png
- **Størrelse**: 1024x1024 piksler
- **Format**: PNG
- **Bakgrunn**: #2E7D32
- **Beskrivelse**: Android adaptive icon

### 4. favicon.png
- **Størrelse**: 48x48 piksler (minimum)
- **Format**: PNG
- **Beskrivelse**: Favicon for web-versjonen

## Anbefalte verktøy

- **Figma**: For å designe ikoner
- **Photoshop/GIMP**: For bildebehandling
- **Online tools**: 
  - https://www.appicon.co/
  - https://makeappicon.com/

## Designforslag

- Bruk organisasjonens logo eller symbol
- Bruk grønn farge (#2E7D32) som hovedfarge
- Inkluder natur-elementer (fjell, trær, stier) for å reflektere organisasjonens fokus
- Hold designet enkelt og gjenkjennelig i små størrelser

## Midlertidig løsning

Hvis du ikke har bilder klar ennå, kan du:
1. Bruke Expo's standard-ikoner ved å kjøre `expo prebuild`
2. Opprette enkle placeholder-bilder med en grønn bakgrunn og tekst
3. Bruke organisasjonens eksisterende logo fra nettstedet
