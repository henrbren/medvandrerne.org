# Quick Start Guide

## Rask oppstart

1. **Installer avhengigheter**
   ```bash
   npm install
   ```

2. **Start utviklingsserveren**
   ```bash
   npm start
   ```

3. **Test på enhet**
   - Installer Expo Go-appen på telefonen din
   - Skann QR-koden som vises i terminalen
   - Appen vil laste automatisk

## Hva er inkludert?

✅ Komplett navigasjon med 5 hovedskjermer
✅ All organisasjonsinformasjon fra medvandrerne.org
✅ Interaktiv kalender med aktiviteter
✅ Kontaktinformasjon for alle lokallag
✅ Støttefunksjonalitet (Spleis, VIPPS, bankkonto)
✅ Moderne UI med grønn fargepalett
✅ Norsk språkstøtte

## Neste steg

1. Legg til app-ikoner (se ASSETS_GUIDE.md)
2. Test appen på iOS og Android
3. Tilpass farger/styling etter behov
4. Legg til ekstra funksjoner hvis ønskelig

## Troubleshooting

**Problem**: "Module not found"
- **Løsning**: Kjør `npm install` på nytt

**Problem**: "Expo Go kan ikke koble til"
- **Løsning**: Sørg for at telefonen og datamaskinen er på samme nettverk

**Problem**: Kalender vises ikke
- **Løsning**: Sjekk at `react-native-calendars` er installert: `npm install react-native-calendars`
